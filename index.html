Version History

v1.0.5 (June 2026)

  • Comprehensive Technical Manual: Embedded the finalized Engineering PDF detailing all mathematics, algorithms, and panel definitions. The "Help" button now directly opens this PDF.
  • UI & Release Alignment: Fully synced all internal application versions, changelogs, and credit windows.

v1.0.4 (June 2026)

  • Project Report Engine: Completely rewrote the reporting engine to output full `docx` and `pdf` project reports containing exhaustive theoretical context.
  • Versine Formula Corrections: Hardcoded the true station indexes for chords: [i-1]/[i+1] for 20m, and [i-4]/[i+4] for 80m.
  • Filename Precision: Fixed a start/end chainage text formatting bug in the PDF/Excel exports.

v1.0.3 (May 2026)

  • Fixed results table header sticky depth and layout alignment mismatch.
  • Fixed Excel import parsing and auto-sheet detection robustness.
  • Restored lowering deviation penalty multiplier to 10.0x.
  • Hidden boundary chord readings (V20/V80) from dashboard metrics and user-facing tables.
  • Removed redundant lock column from results panel.

v1.0.2 (May 2026)

  • Added custom max lift/lower station overrides.
  • UI alignment fixes.

v1.0.1 (May 2026)

  • Fixed constraints and metrics calculation.

v1.0.0 (May 2026)

  • Initial release of TrackLift Pro.
  • Projected Gradient Descent (PGD) optimization solver for rail level smoothing.
  • Constraint compliance tracking for 20m and 80m chord versines.
  • Sliding window Standard Deviation checks (200m block).
  • Dynamic profile graphs with horizontal panning, window controls, and vertical exaggeration slider.
  • Excel and PDF report export integration.
